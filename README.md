V2EX
====

V2EX unofficial App
时不时的上V2EX看看有啥好玩的东西，却发现缺少一个客户端，看到Lee做了个IOS的客户端，于是也为Android端做一个App。功能比较简单，简单的抓取网页，用两个业余晚上基本完成。个人认为UI还不错，如果有喜欢的同学进一步开发，欢迎fork，可以一起开发更多的功能如登录等…当然了针对于喜欢V2EX的童鞋。
